PythonJamaica is a user group for Python users based in Jamaica.  Whether you’re a newbie or a veteran you’re welcome. The group aims to provide a space for local Python developers, learners, employers, and others to interact.  All skill levels are welcome, as long as you have an interest in the Python programming language.

Areas of Special Interest

As Python is used for a wide number of applications the group covers a variety of topics. These are some areas that we are always looking for speakers

Web and Mobile Application Development

Web Application Frameworks, Content Management Systems, Enterprise Resource Management Platforms, Mobile Application Development tools and more. We're interested in learning how to take advantage of existing tools and also give back to the the large Python ecosystem.

3D Animation

The flagship open source 3D animation suite called Blender is written in Python and can be extended using Python scripting. There is also a lot of scope for automation of rendering pipelines with Python.

Devops

DevOps is about agile and scalable system administration, tools such as Ansible, SaltStack and Fabric help to make this possible with Python.
